GLOSARIO DE HTML Y CSS

Introduccion

Esta pagina web creada y diseñada por estudiantes de 4to año esta dirigida a estudiantes de cursos anteriores para que lo puedan utilizar para guiarse y saber las bases de html y css que suelen ser complicadas.

HTML

HTML(HyperText Markup Language) es el lenguaje standard para las paginas webs este es un idioma que funciona para la interacctuacion y creacion de paginas webs. Este lenguage permite crear y diseñar paginas webs a nuestro gusto. Pero aqui venimos ver un glosario sobre lo basico de HTML.

css

CSS(Cascading Style Sheets) es un complemento de HTML que permite darle toda la magia y diversion a las paginas webs, sin esto todas las paginas serian aburridas ya que son las que le otorgan caracteristicas propias y unicas a cada objeto de nuestra pagina web.
